# PROJECT NAME 
 Rentit

# PROJECT DESCRIPTION

Rentit is a secure and trustworthy platform where people can rent items to people  who want a quick use of those items, systems for security trust and verification will be set in place. 

Rentit can also serve businesses who need bigger hardware for specific purposes like renting cement tanks for mixing cement ,renting trailers for digging up roads
